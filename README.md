# Online_Examination_System
**Welcome to my Online Examination System project**, a platform designed to automate and simplify the process of conducting exams in educational institutions. The system allows students to take exams online while administrators and teachers can manage question papers, schedule tests, and evaluate results automatically. It minimizes manual work, reduces errors, and enhances the efficiency, flexibility, and security of the examination process.

# Features 🎓
* **User Registration & Login:** Secure authentication system for students, teachers, and administrators.
* **Exam Management:** Admin and teachers can create, update, schedule, or delete online examinations.
* **Question Bank:** Supports adding multiple-choice, true/false, and descriptive questions.
* **Online Test Interface:** Students can attend exams online with a simple and user-friendly interface.
* **Automatic Evaluation:** Objective-type questions are evaluated instantly with automatic score generation.
* **Result Management:** Students can view their exam results and performance reports online.
* **Timer Functionality:** Exams include countdown timers for better exam control and management.
* **Admin Dashboard:** Manage users, exams, questions, results, and student records efficiently.
* **Secure Examination System:** Reduces malpractice with authentication and controlled exam access.
* **Responsive Design:** Accessible on desktops, tablets, and mobile devices.

# Technologies Used 💻

* **Java Programming:** Used for backend development, business logic implementation, Servlets, and JPA entity management.
* **JSP (Java Server Pages):** Used to create dynamic and interactive web pages integrated with Java backend components.
* **Servlets:** Handles HTTP requests, user authentication, exam management, and result processing on the server side.
* **MySQL Database Server:** Used for storing user records, exams, questions, answers, and results in a relational database.
* **JDBC (Java Database Connectivity):** Used to connect the Java application with the MySQL database and perform CRUD operations.
* **JPA (Java Persistence API):** Used for Object Relational Mapping (ORM) and database entity management.
* **HTML:** Used to structure webpages and examination forms.
* **CSS:** Used for styling and responsive user interface design.
* **JavaScript:** Used for form validation, timers, interactive navigation, and dynamic functionalities.
* **Bootstrap:** Used for responsive layouts and modern UI components.

# Installation 📥

1. Install XAMPP to run Apache and MySQL server.
2. Install a text editor like Notepad++, or any editor of your choice.
3. Download the zip file named "Online_Examination_System".
4. Extract the zip file using WinRAR or any extraction tool.
5. Copy the folder named "Online_Examination_System".
6. Go to the XAMPP installation directory (usually C:\xampp\htdocs).
7. Paste the "Online_Examination_System" folder inside the htdocs folder.
8. Start XAMPP and run Apache and MySQL.
9. Open your browser and go to: http://localhost/phpmyadmin.
10. Create a new database named: Database.
11. Click "Import", then select and import the file Database.sql from the database folder.
12. In your browser, go to: http://localhost/Online_Examination_System.
